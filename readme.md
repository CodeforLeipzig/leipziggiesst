# giessdenkiez.de  

Source repo for [giessdenkiez.de](https://www.giessdenkiez.de)

To get the map tiles loaded you need to create a `.env` file and add the following:

```plain
API_KEY=************
AUTH0_DOMAIN=*******
AUTH0_CLIENT_ID=****
AUTH0_AUDIENCE=*****
```

Start the project via  

```plain
npm start
```
